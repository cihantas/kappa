# Kappa

Kappa is a client library for the Twitch REST API.

- [Installation](#installation)
- [Usage](#usag)
- [License](#license)

## Installation

```bash
$ go get github.com/cihantas/kappa
```
And then import the package.

```go
import "github.com/cihantas/kappa/twitch"
```
